# Mentoria PCAV — Site de Vendas

Landing page da Mentoria PCAV de Lucas Vilaronga.
Mentoria individual para vendedores do Mercado Livre focada em leitura de dados.

**Site:** https://olucasvilaronga.com.br

---

## Rodando localmente

```bash
npm install
npm run dev
```

## Deploy

```bash
python3 deploy.py
```

Faz o build e sobe automaticamente via FTP para a Hostinger.
